# ECHO RIFT: OVERTURE 7.0.2 — PRISM

**3초를 설계하고, 과거의 나와 같은 적을 연다.**

최근 약 3초의 이동·조준·사격·대시를 고정해 과거의 자신과 교차 사격하는 오프라인 브라우저 액션 로그라이트입니다. CROSSFIRE의 위상 균열, FORECAST의 최종 위험 공개, ONRAMP의 4단계 기본 훈련과 선택형 고급 훈련, OVERTURE의 보스 인트로 전투 정지는 그대로 유지합니다.

OVERTURE 7.0.0 PRISM의 기반 변경은 **탄막 장면의 렌더링 성능 회복**입니다. 네온 글로우 비주얼은 유지하되 hot path의 Canvas `shadowBlur`를 사전 렌더 글로우 스프라이트로 교체하고, 자동 품질 강등은 물체 수가 아니라 실측 프레임타임을 우선 근거로 삼습니다. 이후 7.0.1은 로컬 CC0 에셋 레이어를, 7.0.2는 첫 런 코치를 더했으며 신규 강화·전투 수치 밸런스·온라인 텔레메트리는 변경하지 않았습니다.

## 7.0.2 첫 런 코치 패스

- **전투 중 3단계 코치**: 첫 일반 런에서만 `기록하기 → 잔향 호출 → 같은 적 협공`을 짧은 비차단 패널로 안내합니다.
- **실제 행동 기반 완료**: 코치는 REC 버퍼의 사격 기록, 실제 잔향 호출, 위상 균열 카운터를 읽어 단계가 넘어갑니다. 문자열만 표시하고 끝나는 도움말이 아닙니다.
- **반복 노출 방지**: 성공, 숨김, 시간 초과 중 하나가 발생하면 저장에 `fieldCoachSeen`을 기록합니다. 메뉴의 기본/고급 훈련은 기존 단계 수를 유지합니다.
- **검증 추가**: `scripts/verify-first-run-coach.mjs`가 Playwright로 코치 시작, 단계 전환, 위상 균열 완료, 숨김, 훈련 모드 비노출, 3개 뷰포트 레이아웃을 확인합니다.

## 7.0.1 에셋 패스

- **CC0 로컬 에셋 레이어**: Kenney UI Audio, Sci-Fi Sounds, Space Shooter Extension에서 고른 작은 CC0 부분 집합을 저장소에 포함했습니다. 출처와 라이선스는 `THIRD_PARTY_NOTICES.md`와 `assets/vendor/kenney/**/License.txt`에 기록했습니다.
- **샘플 기반 피드백 보강**: 기존 절차 합성음은 유지하고, UI 선택/리롤/경로 선택/사격/위상 균열/보호막 파괴/폭발/보스 펄스에 짧은 OGG 샘플을 낮은 레이어로 겹칩니다.
- **가벼운 효과 스프라이트**: Space Shooter Extension의 작은 효과 PNG를 기존 particle pool 안에서 충격 구름과 위상 플레어로만 사용합니다. 저품질·감소 모션에서는 기존 규칙대로 생략됩니다.
- **오프라인 유지**: 새 오디오·이미지·라이선스 파일은 서비스워커 캐시와 체크섬에 포함되어, 설치 후 외부 네트워크 없이 동작합니다.

## 7.0 핵심 변경

- **글로우 스프라이트 렌더링**: 적탄, 플레이어 탄환, 입자의 네온 번짐을 `ctx.shadowBlur` 대신 미리 구운 글로우 스프라이트 `drawImage`로 그립니다.
- **핫패스 shadowBlur 제거**: 전투 탄막/입자 draw 경로에서 `ctx.shadowBlur` 쓰기를 제거했습니다. 6.8의 화살·마름모·고리 형상과 전투 팔레트는 유지됩니다.
- **프레임타임 기반 품질 보호**: 자동 품질 강등은 실측 프레임 시간과 긴 프레임 점수를 기준으로 작동합니다. 강등 후 런 중 재상향하지 않는 sticky 계약은 유지합니다.
- **렌더 벤치 QA 훅**: `__echoRiftQA.renderBenchmark()`가 적탄/입자 수를 고정한 장면에서 평균·최대 프레임타임, 글로우 패스 수, shadow blur 사용 횟수를 반환합니다.
- **유지된 경계**: 6.11 키 리매핑, 강화 카드 선택/잠금 구조, 부분 리롤 계측, 6.10 저장/import 하드닝, 서비스워커 요청별 폴백은 그대로 유지합니다.

## 6.11 핵심 변경

- **키보드 리매핑**: 설정 화면에서 이동, 조준, 사격, 대시, 잔향 호출, 리롤, 일시정지 키를 변경하고 기본값으로 되돌릴 수 있습니다.
- **강화 카드 접근성**: 카드 전체가 버튼이던 구조를 `article` 컨테이너와 선택/잠금 형제 버튼으로 분리해 중첩 인터랙티브 요소를 제거했습니다.
- **부분 리롤 계측**: 런마다 리롤 횟수, 리롤당 평균 잠금 카드 수, 최대 잠금 카드 수, 리롤 직후 선택 횟수, 2/3단계 시너지 도달 시점을 로컬 상태와 최근 런 요약에 기록합니다.
- **90초 행동 검증**: `scripts/verify-6.11-control.mjs`가 실제 설정 UI 리매핑, 카드 구조, 리롤 계측, 장시간 전투 루프를 Playwright로 확인합니다.
- **유지된 경계**: 물리 게임패드 리매핑, 터치 버튼 위치 편집, 저사양 실기기 성능, 오디오 청감 검증은 별도 하드웨어 확인 항목입니다.

## 6.10 핵심 변경

- **행동 기반 회귀 테스트**: 카드 잠금/부분 리롤, 저장 가져오기 왕복과 거부 케이스, 보스 인트로 정지, 경로 예고/실제 수치 일치를 Playwright로 실제 실행합니다.
- **저장 가져오기 하드닝**: save/settings를 화이트리스트 정규화하고, 임시 키 쓰기와 읽기 검증 뒤 실제 키로 커밋합니다. 실패 시 기존 저장으로 롤백합니다.
- **가져오기 되돌리기**: 설정 화면에서 마지막 가져오기 전 백업을 한 번 복원할 수 있습니다.
- **서비스워커 폴백 수정**: `index.html` 폴백은 문서 탐색 요청에만 적용하고, 스크립트·CSS·이미지·manifest 실패는 실제 실패로 드러냅니다.
- **접근성 보강**: 브라우저 확대를 막던 `user-scalable=no`를 제거했습니다. 캔버스와 터치 조작 영역의 `touch-action`은 유지합니다.

## 6.9 핵심 변경

- **카드 잠금·부분 리롤**: 강화 카드별 잠금 토글을 추가했습니다. 잠긴 카드는 id와 희귀도를 유지하고, 리롤은 잠기지 않은 카드만 새로 계산합니다.
- **오프라인 저장 내보내기·가져오기**: 진행 저장, 선택적 설정, 런 기록을 `ECHO_RIFT` JSON 봉투로 내보냅니다. 가져오기는 product/schema/payload/checksum/1MB 제한을 검증한 뒤 적용하고, 적용 전 현재 저장을 백업합니다.
- **최근 런 기록**: 승리 또는 사망 시 점수, 구역, 시간, 레벨, 처치, 주력 빌드, 잔향 비중, 위상 균열 요약을 최대 20개까지 저장합니다.
- **6.8 SIGNAL 유지**: 투사체 형상, 색약 팔레트, 고대비 결과 화면, 짧은 가로 메뉴 CTA는 그대로 유지됩니다.

## 실행

- 빠른 실행: 압축을 푼 뒤 `index.html`을 엽니다.
- 설치형 PWA·완전한 오프라인 캐시: Windows는 `start_windows.bat`, macOS/Linux는 `start_mac_linux.sh`를 실행합니다.
- 외부 라이브러리, 계정, 런타임 네트워크 연결이 필요하지 않습니다. 포함된 CC0 오디오·이미지 에셋은 모두 로컬 파일로 제공됩니다.
- ASCENSION 2.0 이후 저장 데이터와 설정을 이어서 읽습니다.

## 6.6 핵심 변경

### 1. 보스 인트로 동안 전체 전투 정지

보스 구역은 **1.6초 타이틀 + 1.25초 실체화**의 두 단계로 시작합니다. 총 2.85초 동안 다음 항목이 모두 멈춥니다.

- 플레이어 이동·사격·대시
- 적과 보스 AI
- 적대 탄환·레이저·충돌 판정
- 플레이어 능력 쿨다운
- 잔향 기록과 재생
- 런 생존 시간

따라서 중앙 타이틀을 읽는 동안 피해를 받을 수 없습니다.

### 2. 보스는 배너가 끝난 뒤 생성

기존에는 보스가 중앙 배너 뒤에서 생성되고 첫 공격 준비를 시작할 수 있었습니다. 이제 인트로 중에는 보스 개체 자체가 존재하지 않습니다.

```text
보스 구역 시작
→ TIME LINE SUSPENDED 1.6초
→ 배너 종료·보스 생성
→ MATERIALIZE 1.25초 · 피해 판정 없음
→ PHASE 1 / COMBAT LIVE
→ 1.05초 첫 공격 유예 뒤 전투
```

이전 구역에서 남은 플레이어·적 탄환과 레이저도 인트로 진입 시 정리해 선제 공격과 잔여 피격을 방지합니다.

### 3. 진행 막대와 정확한 카운트다운

- `NULL ARCHIVIST` 전용 인트로 스타일
- 남은 시간 0.1초 단위 표시
- 보라→분홍→금색 진행 막대
- 인트로 동안 HUD와 터치 조작계를 낮은 명도로 처리
- 작은 세로 화면과 짧은 가로 화면용 전용 크기

### 4. 런 기록에서 연출 시간 제외

타이틀과 실체화의 총 2.85초는 생존 시간, 위협도, 보스 격노 타이머에 포함되지 않습니다. 일시정지하면 현재 단계의 카운트다운도 그대로 멈추며 재개 후 이어집니다. 실체화 중 보스에게 공격을 적중시켜도 피해는 0입니다.

## 유지된 주요 시스템

- 필수 첫 훈련 4단계, 고급 전술 훈련 3단계
- 현재 몸과 잔향의 교차 공격으로 `위상 균열` 발동
- 경로 선택 전에 최종 적·보상 배율 공개
- 입력 순간 3초 스냅숏 고정
- 키보드·게임패드·터치 잔향 미리보기
- 마지막 투사체와 지속 피해까지 끝난 뒤 잔향 리포트 확정
- 63개 강화, 8개 진화, 6단계 희귀도, 12개 영구 연구, 6개 균열 경로
- 승리 후 끝없는 시간선
- PWA·오프라인 실행·기존 저장 데이터 호환
