# ECHO RIFT: OVERTURE 6.9 — INTENT

**좋은 선택지는 붙잡고, 진행 데이터는 직접 보관한다.**

최근 약 3초의 이동·조준·사격·대시를 고정해 과거의 자신과 교차 사격하는 오프라인 브라우저 액션 로그라이트입니다. CROSSFIRE의 위상 균열, FORECAST의 최종 위험 공개, ONRAMP의 4단계 기본 훈련과 선택형 고급 훈련, OVERTURE의 보스 인트로 전투 정지는 그대로 유지합니다.

OVERTURE 6.9는 **선택 통제와 데이터 안전**에 집중한 버전입니다. 강화 선택 화면에서 남기고 싶은 카드를 잠근 뒤 나머지만 부분 리롤할 수 있고, 진행 저장을 체크섬이 포함된 JSON 파일로 내보내거나 가져올 수 있습니다. 최근 런 기록은 최대 20개 요약만 로컬에 보관합니다. 신규 강화·수치 밸런스·텔레메트리는 변경하지 않았습니다.

## 6.9 핵심 변경

- **카드 잠금·부분 리롤**: 강화 카드별 잠금 토글을 추가했습니다. 잠긴 카드는 id와 희귀도를 유지하고, 리롤은 잠기지 않은 카드만 새로 계산합니다.
- **오프라인 저장 내보내기·가져오기**: 진행 저장, 선택적 설정, 런 기록을 `ECHO_RIFT` JSON 봉투로 내보냅니다. 가져오기는 product/schema/payload/checksum/1MB 제한을 검증한 뒤 적용하고, 적용 전 현재 저장을 백업합니다.
- **최근 런 기록**: 승리 또는 사망 시 점수, 구역, 시간, 레벨, 처치, 주력 빌드, 잔향 비중, 위상 균열 요약을 최대 20개까지 저장합니다.
- **6.8 SIGNAL 유지**: 투사체 형상, 색약 팔레트, 고대비 결과 화면, 짧은 가로 메뉴 CTA는 그대로 유지됩니다.

## 실행

- 빠른 실행: 압축을 푼 뒤 `index.html`을 엽니다.
- 설치형 PWA·완전한 오프라인 캐시: Windows는 `start_windows.bat`, macOS/Linux는 `start_mac_linux.sh`를 실행합니다.
- 외부 라이브러리, 외부 음원, 계정, 네트워크 연결이 필요하지 않습니다.
- ASCENSION 2.0 이후 저장 데이터와 설정을 이어서 읽습니다.

## 6.6 핵심 변경

### 1. 보스 인트로 동안 전체 전투 정지

보스 구역은 **1.6초 타이틀 + 1.25초 실체화**의 두 단계로 시작합니다. 총 2.85초 동안 다음 항목이 모두 멈춥니다.

- 플레이어 이동·사격·대시
- 적과 보스 AI
- 적대 탄환·레이저·충돌 판정
- 플레이어 능력 쿨다운
- 잔향 기록과 재생
- 런 생존 시간

따라서 중앙 타이틀을 읽는 동안 피해를 받을 수 없습니다.

### 2. 보스는 배너가 끝난 뒤 생성

기존에는 보스가 중앙 배너 뒤에서 생성되고 첫 공격 준비를 시작할 수 있었습니다. 이제 인트로 중에는 보스 개체 자체가 존재하지 않습니다.

```text
보스 구역 시작
→ TIME LINE SUSPENDED 1.6초
→ 배너 종료·보스 생성
→ MATERIALIZE 1.25초 · 피해 판정 없음
→ PHASE 1 / COMBAT LIVE
→ 1.05초 첫 공격 유예 뒤 전투
```

이전 구역에서 남은 플레이어·적 탄환과 레이저도 인트로 진입 시 정리해 선제 공격과 잔여 피격을 방지합니다.

### 3. 진행 막대와 정확한 카운트다운

- `NULL ARCHIVIST` 전용 인트로 스타일
- 남은 시간 0.1초 단위 표시
- 보라→분홍→금색 진행 막대
- 인트로 동안 HUD와 터치 조작계를 낮은 명도로 처리
- 작은 세로 화면과 짧은 가로 화면용 전용 크기

### 4. 런 기록에서 연출 시간 제외

타이틀과 실체화의 총 2.85초는 생존 시간, 위협도, 보스 격노 타이머에 포함되지 않습니다. 일시정지하면 현재 단계의 카운트다운도 그대로 멈추며 재개 후 이어집니다. 실체화 중 보스에게 공격을 적중시켜도 피해는 0입니다.

## 유지된 주요 시스템

- 필수 첫 훈련 4단계, 고급 전술 훈련 3단계
- 현재 몸과 잔향의 교차 공격으로 `위상 균열` 발동
- 경로 선택 전에 최종 적·보상 배율 공개
- 입력 순간 3초 스냅숏 고정
- 키보드·게임패드·터치 잔향 미리보기
- 마지막 투사체와 지속 피해까지 끝난 뒤 잔향 리포트 확정
- 63개 강화, 8개 진화, 6단계 희귀도, 12개 영구 연구, 6개 균열 경로
- 승리 후 끝없는 시간선
- PWA·오프라인 실행·기존 저장 데이터 호환
