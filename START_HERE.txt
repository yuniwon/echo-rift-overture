ECHO RIFT: OVERTURE 6.11 — 빠른 시작

1. 압축파일을 풉니다.
2. index.html을 더블클릭하면 바로 플레이할 수 있습니다.
3. 설치형 PWA와 완전한 오프라인 캐시를 사용하려면:
   - Windows: start_windows.bat
   - macOS / Linux: start_mac_linux.sh

6.11 변경
- 설정 화면에서 이동·조준·사격·대시·잔향·리롤·일시정지 키를 다시 지정할 수 있습니다.
- 강화 카드는 선택 버튼과 잠금 버튼이 분리되어 키보드와 스크린리더 구조가 더 명확합니다.
- 부분 리롤은 런 기록에 리롤 횟수, 평균 잠금 카드 수, 리롤 직후 선택 횟수를 남깁니다.
- verify-6.11-control은 설정 리매핑, 카드 구조, 리롤 계측, 90초 전투 루프를 실제 브라우저에서 확인합니다.

6.10 변경
- Playwright 행동 회귀 테스트가 카드 잠금·부분 리롤, 저장 가져오기, 보스 인트로 정지, 경로 예고 일치를 실제 브라우저에서 확인합니다.
- 진행 가져오기는 화이트리스트 정규화와 임시 키 검증 뒤 커밋하며, 실패하면 기존 저장을 복원합니다.
- 설정 화면에서 마지막 가져오기 전 백업을 되돌릴 수 있습니다.
- 서비스워커는 문서 탐색 요청에만 index.html로 폴백합니다.
- 브라우저 확대를 허용합니다.

6.9 변경
- 강화 선택 화면에서 카드를 잠그면 리롤 시 잠긴 카드는 유지되고 나머지만 다시 계산됩니다.
- 설정 화면의 데이터 섹션에서 진행 저장을 JSON 파일로 내보내거나 가져올 수 있습니다.
- 최근 런 기록은 최대 20개 요약만 로컬에 저장됩니다.

보스전 변경
- SECTOR 06과 이후 끝없는 보스 구역에서는 1.6초 동안 TIME LINE SUSPENDED가 표시됩니다.
- 타이틀 뒤 보스가 생성되지만 1.25초 동안 MATERIALIZE 상태로 피해와 공격이 잠깁니다.
- 총 2.85초 동안 플레이어·적·탄환·레이저·능력 쿨다운·런 타이머가 모두 멈춥니다.
- COMBAT LIVE 뒤에도 첫 공격까지 최소 1.05초의 유예가 있습니다.
- 일시정지하면 현재 보스 인트로 단계의 카운트다운도 멈춥니다.

핵심 조작
- 이동: WASD / 왼쪽 스틱 / 왼쪽 가상 스틱
- 조준·사격: 마우스 / 방향키+J / 오른쪽 스틱 / 오른쪽 가상 스틱
- 위상 대시: Space·Shift / 게임패드 A·LB / 대시 버튼
- 잔향: E·Q / 게임패드 B·RB / 잔향 버튼
- 강화 선택: 1–6 / 클릭 / D-Pad+A / 탭
- 경로 선택: 1–3 / 클릭 / D-Pad+A / 탭
- 일시정지: Esc / 메뉴 버튼
- 성능 정보: F3

키보드 기본값은 설정 > 키보드 입력에서 변경하거나 기본값으로 복구할 수 있습니다.

기본 훈련은 4단계이며, 교차 사격과 결과 판독은 메인 메뉴의 고급 전술 훈련에서 연습할 수 있습니다.

ASCENSION 2.0 이후의 기존 저장 데이터와 설정을 그대로 읽습니다.
