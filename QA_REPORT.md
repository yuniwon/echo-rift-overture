## 수동 확인 필요 (미검증 한계)

다음은 실제 브라우저에서 사용자 확인이 필요합니다.

- 흑백/색약 팔레트에서 100개 이상 투사체 장면의 소유권 판독과 긴 프레임 회귀
- 결과 화면 본문 대비 4.5:1 측정 및 320×568 첫 화면 노출
- 667×375 / 740×360 / 812×375 가로 화면에서 스크롤 없는 시작 버튼 노출, Safe Area 침범 여부
- 설정 저장 후 재시작 유지
- 실제 모바일/저사양 단말 성능, 물리 게임패드, 장시간 오디오
