## 6.8.0 — OVERTURE / SIGNAL

### 전투 가독성: 투사체 소유권 형상

- 현재 몸 탄환을 화살(arrow) 머리로 렌더링
- 잔향 탄환을 마름모(diamond) 머리로 렌더링
- 적 탄환을 속이 빈 고리(ring) + 작은 코어로 렌더링
- 형상은 속도 방향으로 회전하며, 색상을 제거해도(흑백) 세 소유권 구분 가능
- 형상 머리는 배치 렌더링을 유지(소유권 그룹당 단일 fill/stroke, 탄환별 save/rotate 없음)
- `투사체 형상 구분` 설정으로 끄면 기존 원형 머리로 복귀
- 형상은 성능 모드(quality 0)에서도 유지, 저품질에서는 장식 꼬리만 축소

### 색약 보정 팔레트

- `전투 색상 팔레트` 설정 추가: 기본 / 적록 보정 / 청황 보정 / 고대비 단색

- 기본 팔레트에서는 기존 탄환별 색을 보존(시각 회귀 없음)
- 단색 모드는 형상으로만 소유권을 구분
